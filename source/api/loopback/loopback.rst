PATCH
---------------------------------------

请求结构
+++++++++++++++++++++++++++++++++++++++


请求参数
+++++++++++++++++++++++++++++++++++++++

==============  =========  ==================  ====================
参数名称         是否必选    参数类型及范围        参数含义
==============  =========  ==================  ====================
Description     否         string              接口描述
Address         是         点分十进制整数        接口IP地址
Member          否         1、2                项目
==============  =========  ==================  ====================

注意事项
+++++++++++++++++++++++++++++++++++++++
堆叠系统时，Member必选。单机系统不需要配置。

调用样例
+++++++++++++++++++++++++++++++++++++++
调用样例如下::

 {
   "Description": "",
   "Addresses": [
     {
       "Address": "1.1.1.1/24",
       "Member": 1
     }
   ]
 }
